# Welcome MeowReader

* 方便、快速瀏覽 Markdown 檔案
* 支持 LaTeX 數學
* Project Github：[MoewReaDer](https://github.com/music1353/MeowReaDer)



## 操作說明

1. 滑鼠移動到右上角的齒輪
2. 選擇欲瀏覽的資料夾位置（會將選定資料夾以下的所有md檔都匯入）
3. 享受文件 =v=
